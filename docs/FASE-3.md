# Fase 3 - Seccion beneficios

Se agrego una seccion dedicada a la metodologia de entrenamiento.

## Beneficios

Cards creadas:

- Tecnica
- Mentalidad
- Creatividad
- Disciplina
- Liderazgo
- Confianza

## Cada card incluye

- Icono de `lucide-react`.
- Hover glow cyan.
- Animacion de entrada `card-rise`.
- Efecto scan line.
- Layout responsive: 1 columna mobile, 2 columnas tablet, 3 columnas desktop.

## Archivos

- `frontend/src/app/page.tsx`
- `frontend/tailwind.config.ts`
- `frontend/src/lib/site.ts`

## Verificacion

```bash
npm run build:frontend
npm --prefix frontend run lint
```
